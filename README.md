# Avito_DS-Test[NLP_LLM]
Avito DS Bootcamp 2026 Summer: Решение тестового задания 🚀
Решение задачи информационного поиска (Information Retrieval) по базе статей справки Авито:
по вопросу пользователя нужно найти релевантные статьи и выдать топ-10. Метрика — MAP@10.

📊 Метрика на калибровке (MAP@10): 0.458 (5-fold CV: 0.458 ± 0.023)

🛠 Архитектура решения (Hybrid Retrieval)
Пайплайн на локальных open-source моделях, одноэтапный и дешёвый на инференсе.
Итоговый скор статьи — взвешенная сумма трёх нормированных сигналов:

Лексический поиск: тюнингованный BM25 с лемматизацией русского (pymorphy3) — точные совпадения слов, терминов, номеров.

Семантический поиск: эмбеддинги BAAI/bge-m3 (вектор всей статьи) — смысл при «разрыве словаря», когда запрос и статья про одно, но разными словами.

Чанки: статья режется на куски, берётся лучший (max-pooling) — релевантный абзац не тонет в длинной статье.

Cross-encoder реранкер сознательно не внедрён: на CPU он непрактично медленный, а сильный bi-encoder + чанкинг дали нужное качество дешевле. Подробный разбор этого размена — в notebook.
